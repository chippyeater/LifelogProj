# 任务
你要为一组“真实子事件”批量生成一一对应的错误子事件选项，用于回忆题中的干扰项。

# 上下文
当前活动主题：
`{{ACTIVITY_NAME}}`

输入是一组事件对象 JSON，字段含义：
- `event_name`: 真实子事件名称

输入事件列表如下：
{{EVENTS_JSON}}

# 要求
1. 为每个真实子事件生成且只生成一个错误子事件名称。
2. 输出顺序必须与输入顺序完全一致。
3. 错误子事件必须和对应真实子事件明显不同，但仍然属于日常生活/购物/活动场景中可能出现的事件，并且抽象层级一致。
4. 不要输出原事件名称、同义词、近义词、上下位词、过于相似的表述。
5. 特别避免“弱包含关系”“同品类上下位关系”“同货架相邻品类”导致的歧义。
例如：
- “选购乳制品” 不能配 “挑选饮料”
- “购买水果” 不能配 “挑选苹果”
- “选购蔬菜” 不能配 “购买海鲜”
因为这些选项和正确事件存在包含、邻近或过弱区分关系，容易误导。
6. 每个错误子事件名称为4-6个字。
7. 只返回 JSON 数组，不要返回解释、Markdown、代码块。

# 输出示例
[
  {"event_name":"选购乳制品","distractor_option":"选购水果"},
  {"event_name":"挑选蔬菜","distractor_option":"购买海鲜"}
]

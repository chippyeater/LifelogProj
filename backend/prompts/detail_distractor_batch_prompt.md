# 任务
你要为一组“实体细节题”批量生成与正确物体一一对应的错误物体名，用于回忆题中的干扰选项。

# 上下文
当前活动主题：`{{ACTIVITY_NAME}}`

输入是一组事件对象 JSON，字段含义：
- `event_name`：子事件名称
- `correct_items`：该子事件中的正确物体名列表
- `max_items`：该子事件最多需要生成多少个错误物体名

输入事件列表如下：
{{EVENTS_JSON}}

# 要求
1. 为每个子事件生成一组错误物体名，数量不超过对应的 `max_items`。
2. 输出顺序必须与输入顺序完全一致。
3. 错误物体名必须与对应正确物体明显不同，但抽象层级相同，且仍然属于当前活动主题下可能出现的具体物体。
4. 不要输出正确物体本身、同义词、近义词、品牌变体、规格变体、包装变体。
5. 特别避免弱包含、上下位、同套餐组成造成的歧义。
例如：
- “苹果”不能配“水果”
- “虾仁”不能配“海鲜”
1. 输出物体名的字数为2-4个字。
2. 只返回 JSON 数组，不要返回解释、Markdown、代码块。

# 输出格式
```json
[
  {
    "event_name": "选购乳制品",
    "distractor_options": ["苹果", "面包", "洗发水"]
  },
  {
    "event_name": "挑选蔬菜",
    "distractor_options": ["虾仁", "鸡胸肉", "橙汁"]
  }
]
```
